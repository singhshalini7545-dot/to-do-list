## Quick Start

1. *Clone or Download* the repository
2. *Open* index.html in your web browser
3. *Start using* Productivity Pro immediately!

No installation or setup required - it works directly in your browser.

## Usage

### Adding Tasks
1. Click the "Add Task" button
2. Fill in task details (title, description, priority, category, due date)
3. Save to add to your task list

### Using the Focus Timer
1. Select your preferred mode (Pomodoro, Short Break, Long Break)
2. Click "Start" to begin the timer
3. The app will automatically transition between sessions

### Calendar Navigation
1. Use the arrow buttons to navigate between months
2. Click on any date to view tasks scheduled for that day
3. Days with tasks are marked with colored dots

### Theme Customization
1. Click the "Settings" button
2. Choose from 20 different color themes
3. Your selection is automatically saved
